Conversational Research Paper Recommender

A chatbot for academic paper discovery powered by TF-IDF, topic classification, semantic similarity, and conversational follow-up support.

What this chatbot can do

This system helps users explore research papers in areas such as natural language processing, machine learning, recommender systems, databases, information retrieval, and computer vision. It predicts the topic of your query, retrieves relevant papers, and lets you ask follow-up questions about the recommendations.

TF-IDF + n-grams
Naive Bayes classification
Cosine similarity ranking
Dialogflow memory contexts
FastAPI + Render deployment

Topic Prediction

The chatbot predicts the most likely research topic from your query and uses that as part of the recommendation process.

Paper Recommendation

It ranks papers using semantic similarity plus a topic-based score boost to return relevant recommendations.

Conversational Follow-up

After recommending papers, the chatbot can answer follow-up questions such as “show me more” or “tell me more about paper 1”.

Try questions like these

Recommend papers on recommender systems

Tip: specific queries usually produce better recommendations than very broad queries like “AI” or “machine learning”.

How it works

The backend is deployed with FastAPI and connected to Dialogflow through webhook fulfillment. When a user sends a query, the system cleans the text, predicts a topic label using Multinomial Naive Bayes, computes semantic similarity with candidate papers, and returns the top recommendations. The chatbot also stores the last recommended papers in memory, so users can ask for more details in follow-up turns.

Project Context

This chatbot was developed as a final project in Data Science Applications. It combines data preparation, text feature engineering, classification, clustering, recommendation, deployment, and conversational interaction in one integrated system.
